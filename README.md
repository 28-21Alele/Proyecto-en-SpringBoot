# Kenaro's Store

Kenaro's Store es una aplicación web desarrollada con Spring Boot que simula una tienda digital de videojuegos. Permite visualizar un catálogo de productos, realizar búsquedas, filtrar por categorías, consultar detalles de cada videojuego y gestionar un carrito de compras mediante sesiones.

---

### Catálogo de videojuegos

- Visualización de videojuegos disponibles.
- Portadas personalizadas para cada producto.
- Información detallada de cada videojuego.
- Navegación intuitiva.

### Búsqueda y filtros

- Búsqueda por nombre.
- Filtrado por categoría.
- Combinación de búsqueda y categoría.
- Mensajes cuando no existen resultados.

### Detalle del producto

- Imagen del videojuego.
- Categoría.
- Descripción.
- Precio.
- Stock disponible.
- Agregar al carrito.

### Carrito de compras

- Agregar productos.
- Incremento automático de cantidades.
- Eliminación individual de productos.
- Vaciar carrito completo.
- Cálculo automático de subtotales y total.
- Contador dinámico de productos.
- Persistencia mediante sesión (HttpSession).
